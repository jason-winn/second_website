Airbnb Map Pin Clustering

Analyzing the Globe In Nanoseconds

Map pin clustering involved the algorithmic grouping of Airbnb's millions of pins on a world map. This project was my own from the outset until completion. Performance and scalability were a top priority.

I pre-processed the data using a modified quad-tree algorithm, represented the tree using a hex geo-hash, and assembled the resulting hashes into a trie for low time-complexity access. Client side re-clustering was performed using a modified K-means algorithm.
